# Проект "Сложно сосредоточиться"
## Адаптивный веб-сайт с переключением цветовых тем

### Описание проекта
Одностраничный адаптивный сайт, посвященный проблеме концентрации внимания в современном мире. Проект демонстрирует возможности переключения между темной и светлой темами оформления, а также адаптивную верстку для различных устройств.

### Функциональность
- **Переключение цветовых тем**: темная (неон), светлая (день) и автоматическая (на основе системных настроек)
- **Адаптивный дизайн**: корректное отображение на мобильных устройствах (375px), планшетах (768px) и десктопах (1024px)
- **Интерактивные элементы**: анимированные кнопки переключения тем с визуальной обратной связью
- **Декоративные элементы**: стилизованные уголки, неоновые акценты, индикатор REC для темной темы

### Технологии
- HTML5
- CSS3 (Flexbox, Grid Layout)
- Кастомные CSS-свойства (переменные)
- Медиазапросы для адаптивности
- Поддержка светлой и темной тем через prefers-color-scheme

### Структура проекта
```
project/
├── fonts/               # Шрифты IBM Plex Mono
├── images/              # Изображения и иконки
├── styles/             # CSS файлы
│   ├── dark.css        # Стили для темной темы
│   ├── globals.css     # Глобальные стили и сброс
│   ├── light.css       # Стили для светлой темы
│   ├── style.css       # Основные стили проекта
│   └── variables.css   # CSS переменные
├── scripts/            
│   └── script.js       # Скрипт переключения тем
└── index.html          # Главная страница
```

### Особенности реализации
- **CSS-переменные**: централизованное управление цветами, отступами и размерами
- **Grid-сетка**: сложное позиционирование элементов, особенно в секции с изображениями
- **Адаптивные изображения**: оптимизация отображения картинок на разных разрешениях
- **Кастомные эффекты наведения**: стилизованные рамки для кнопок меню
- **Тени и неоновое свечение**: создание атмосферы в темной теме

### Запуск проекта
1. Склонировать репозиторий
2. Открыть файл `index.html` в любом современном браузере
